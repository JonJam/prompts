# Trophy guide

## Purpose
The prompt instructs an LLM to generate a guide to get the Platinum trophy for a Playstation game.

## Prompt

```
For the specified Playstation 5 game, only use https://www.playstationtrophies.org/ to generate a complete guide to unlock every trophy for that game with the minimum number of playthroughs. The guide should include the full, trophy-by-trophy allocation & links (every trophy listed individually and assigned to which playthrough). Ensure to include references to your sources. 

Game: Final Fantasy VII Remake
```
