# Trophy plan

## Purpose
The prompt instructs an LLM to generate a plan to get the Platinum trophy for a Playstation game.

## Prompt

```
For the specified Playstation 5 game, only use https://www.playstationtrophies.org/ to generate a complete trophy guide to unlock every trophy for that game. Be sure to include references to your sources.

Desired format:
Guide: <Link to guide>

Playthrough: <Name of the playthrough>
<Description of the playthrough>

List of trophies achieved from this playthrough.
<Trophy list formed of name, description and link>

Game: Final Fantasy VII Remake
```
